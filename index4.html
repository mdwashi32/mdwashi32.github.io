Blog Post #1: Street Fighter Alpha 2

Hello everyone! This is my first blog post for Art/CS 108: Introduction to Game Studies.

The purpose of this post is to discuss my experience with a classic arcade game played through Internet Archive.

I chose Street Fighter Alpha 2. When it comes to fighting games, this fan favorite is arguably a classic.

This is not my first time playing the game and I am very familiar with the control scheme and combos.

I didn't expect much from a web browser streaming of the game but I was very surprised. The game ran smoothly and it was almost like actually playing the game on an arcade cabinet.

The primary issue I had with it was setting up controls. By trial and error I was able to figure out how to set inputs through an extremly grainy HUD. Even after that, playing on a keyboard just wasn't the same.

I missed the feel of the ball-top joystick and hard buttons prominent in arcades. Something behind the nostalgia behind using those controls makes me feel like this browser version is missing something.

Other than the controls, again, the game move smoothly. There was no lag or graphical error, and it was as if (minus the controls) I was standing right in front of an arcade cabinet.

Internet Archive definitely achieved its goal of perserving a big piece of history.


Glitch Project - "Clouds"

The objective of this project was to use adobe photoshop and illustrator to create an image and then "glitch" the image by editing its code.

The title of this piece is a play on the term cloud.

In a way, technology is altering the way we perceive the natural world.
